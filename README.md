# E-Commerce Product Listing

Building a product browsing app for the Leegality frontend assessment. It fetches data from [DummyJSON](https://dummyjson.com/docs/products) and has two screens: a listing page with filters and pagination, and a product detail page.

The listing page will show a filter sidebar (category, price range, brand) and a product grid. Each card shows image, title, price, and rating. Clicking a card opens the detail page with full product info and a back button. Filters should work together and stay applied when navigating back.

Built with React, TypeScript, Vite, React Router, Tailwind, and Axios.
